# Northeastern Science Magazine



### Raisa Bhuiyan • 2023 - 2025
1. Test all of your code as you go—it’s not fun to write tests all at once when you have a deadline approaching (and this is coming from a person who enjoys testing)
2. It’s okay not to know everything because that’s why we’re here. If we knew everything about CS we wouldn’t enjoy it; the fun comes from figuring out something for the first time
3. Don’t forget to have fun! You’re also here to make friends so say hi to the people on your team. Hang out with each other once in a while! Doing work is more fun and more doable if you have someone to do the work with
